# Kenya Input-Output Model

Prototype Leontief input-output model for sectoral analysis of Kenya's economy,
built with real data from the **KNBS Economic Survey 2025**.

## What it does
- Estimates inter-sectoral production linkages across 8 sectors
- Computes output multipliers and backward/forward linkages
- Simulates demand shock scenarios (e.g. 10% manufacturing expansion)

## Data Source
Kenya National Bureau of Statistics (2025). *Economic Survey 2025*. Tables 2.1, 2.4, 2.6.

## Usage
Open `kenya_io_model.Rmd` in RStudio and click **Knit → HTML**.

Requires: `ggplot2`, `reshape2`
